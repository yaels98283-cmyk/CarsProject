# CarRental Management System 🚗
**A full-stack solution for vehicle leasing and fleet management.**

## 💡 Overview
This project is a comprehensive Car Rental Management System designed to provide a seamless experience for both administrators and customers. It features a robust backend built with **C#** and **ASP.NET Core**, and a dynamic, responsive frontend developed with **Angular**.

## 🚀 Key Features
*   **Advanced Car Filtering:** Search and filter vehicles based on specific criteria to find the perfect match.
*   **Full CRUD Operations:** Complete management of car inventory and customer data.
*   **Database Integration:** Persistent data storage using **SQL Server**.
*   **Full-Stack Architecture:** Clear separation between the Client (Angular) and the Server (Web API).

## 🛠 Tech Stack
### Frontend
*   **Framework:** Angular
*   **Language:** TypeScript
*   **Styling:** Bootstrap / CSS
*   **Features:** Component-based architecture, Services for API communication.

### Backend
*   **Framework:** ASP.NET Core Web API
*   **Language:** C#
*   **ORM:** Entity Framework Core
*   **Database:** Microsoft SQL Server

## 📁 Project Structure
*   `ProjectYAEL/` - The Angular client application.
*   `CarsSimi/` - The C# Web API server-side logic.
